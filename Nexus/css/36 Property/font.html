CSS Font Property

The font property in CSS is a shorthand property used to set multiple font-related properties in a single declaration. It can include font-style, font-variant, font-weight, font-size, line-height, and font-family.

Font Property Syntax

font: [font-style] [font-variant] [font-weight] [font-size] [line-height] [font-family];

Font Style

Specifies the style of the font. Common values include:

normal | italic | oblique

Font Variant

Specifies whether the text is displayed in small-caps. Common values include:

normal | small-caps

Font Weight

Specifies the weight (thickness) of the font. Common values include:

   
normal | bold | bolder | lighter | 100 | 200 | 300 | 400 | 500 | 600 | 700 | 800 | 900

Font Size

Specifies the size of the font. You can use absolute or relative units. Examples include:

12px | 1em | 100% | 2rem

Line Height

Specifies the height of a line of text. It can be a number, length , or percentage. Examples include:

normal | 1.5 | 20px | 150%

Font Family

Specifies the typeface for the text. You can list multiple font families as a fallback mechanism. Examples include:

Arial, sans-serif | "Times New Roman", serif | "Courier New", monospace

Examples

Here are some examples of how to use the font property in CSS:

   
body {
    font: italic small-caps bold 16px/1.5 "Arial", sans-serif;
}   
h1 {
    font: normal normal 700 32px/1.2 "Times New Roman", serif;
}
