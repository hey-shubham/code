Introduction to CSS

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML or XML.

CSS allows you to control the layout, colors, fonts, and overall visual appearance of web pages. It separates content from design, making it easier to maintain and update the look of a website.

Types of CSS

  • Inline CSS
  • Internal CSS
  • External CSS

Inline CSS

Inline CSS is applied directly to an HTML element using the "style" attribute.

This is a red paragraph.

Internal CSS

Internal CSS is defined within a style tag in the head section of an HTML document.

External CSS

External CSS is written in a separate .css file and linked to the HTML document using a link tag.

CSS Syntax

CSS rules consist of a selector and a declaration block. The selector specifies which HTML elements the rule applies to, and the declaration block contains one or more declarations separated by semicolons.

selector{  
    property: value;  
}

Example

h1 {
    color: blue;
    font-size: 24px;
}

p{
    line-height: 1.5;
    margin-bottom: 10px;
}

CSS Selectors

CSS selectors are used to select the HTML elements you want to style. Some common types of selectors include:

  • Element Selector
  • Class Selector
  • ID Selector
  • Attribute Selector
  • Pseudo-class Selector

Element selector

Selects all elements of a specific type.

 
p {
    color: red;
}

Class selector

Selects all elements with a specific class attribute.

.my-class {
    background-color: yellow;
}

ID selector

Selects a single element with a specific ID attribute.

#my-id {
    font-weight: bold;
}

Attribute selector

Selects elements based on the presence or value of an attribute.

[attribute] {
    /* styles */
}

[attribute="value"] {
    /* styles */
}

Pseudo-class selector

Selects elements based on their state or position.

a:hover {
    text-decoration: underline;
}

li:first-child {
    font-weight: bold;
}

CSS Box Model

The CSS box model describes the rectangular boxes generated for elements in the document tree. It consists of four areas:

  • Content
  • Padding
  • Border
  • Margin

Understanding the box model is essential for controlling layout and spacing in web design.
