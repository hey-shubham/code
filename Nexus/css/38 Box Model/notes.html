Box Model

The CSS Box Model is a fundamental concept in web design and layout. It describes the rectangular boxes generated for elements in the document tree and consists of the following areas:

  1. Content: This is the innermost area where text and images appear. The size of this area can be controlled using properties like width and height.

  2. Padding: This area surrounds the content and provides space between the content and the border. Padding can be set using properties like padding-top, padding-right, padding-bottom, and padding-left, or the shorthand padding.

  3. Border: The border wraps around the padding (if any) and content. You can customize the border's width, style, and color using properties like border-width, border-style, and border-color, or the shorthand border.

  4. Margin: This is the outermost area that creates space between the element and its neighboring elements. Margins can be set using properties like margin-top, margin-right, margin-bottom, and margin-left, or the shorthand margin.

Understanding the Box Model is crucial for creating layouts that look good and function well across different devices and screen sizes.

When we have 2 options for border, it will be allocated to the top and bottom. Border-left and border-right will take 50% each.
