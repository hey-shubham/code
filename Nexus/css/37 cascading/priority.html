CSS Priorities Properties

When multiple CSS rules apply to the same element, the order of the properties can affect which styles are applied. The following rules determine the priority of CSS properties:

  1. Inline styles (styles applied directly to an element using the style attribute) have the highest priority.
  2. ID selectors have higher priority than class selectors, attribute selectors, and pseudo-classes.
  3. Class selectors, attribute selectors, and pseudo-classes have higher priority than element selectors and pseudo-elements.
  4. Element selectors and pseudo-elements have the lowest priority.
  5. If two rules have the same specificity, the one that appears last in the CSS file will take precedence.

!important

!important can be added to a CSS property to give it the highest priority, overriding any other conflicting rules, regardless of specificity or order.

  1. id
  2. attribute
  3. class
  4. child and descendant
  5. element
