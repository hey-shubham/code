## Notes on TypeScript

- **Type Safety**: TypeScript adds static typing to JavaScript, helping catch errors at compile time.

- **Type Annotations**: You can explicitly define the types of variables, function parameters, and return values.

- **Interfaces**: TypeScript allows you to define interfaces to describe the shape of objects, making it easier to work with complex data structures.

## Installation

To install TypeScript, you can use npm (Node Package Manager). Run the following command in your terminal:

```bash
npm install -g typescript@latest
```

## Important Commands

- **Compile TypeScript**: To compile a TypeScript file to JavaScript, use the following command:

```bash
tsc filename.ts
```

## configuration command

```bash
tsc --init
```
## Real-Time Compilation

To compile TypeScript files in real-time, you can use the `--watch` flag:

```bash
tsc --watch
```
